# IT Support Ticket App

A web-based IT support ticket management system built with vanilla JavaScript.

## Features
- Create support tickets with priority levels
- Update ticket status (Open, In Progress, Resolved)
- Delete tickets
- Persistent storage using localStorage
- Modern, responsive UI

## Technologies
- HTML
- CSS
- JavaScript
